# Критерии оценки (черновик)

Оценка складывается из:
- Регулярность и дисциплина (репозиторий, коммиты, структура) - 15%
- Корректность пайплайна ETL (E/T/L, повторный запуск, идемпотентность) - 25%
- Качество данных и проверки (DQ, тесты, отчёт) - 20%
- Аналитика и визуализация (ноутбук/дашборд, выводы) - 20%
- Документация (Implementation Plan, Data Contract, README) - 10%
- Осознанное использование LLM (лог, проверка, отсутствие "магии") - 10%

Бонус (+5%):
- CI (линтер/тесты)
- docker-compose, чтобы "одной командой поднять всё"
- дополнительные источники данных (2+ системы) и грамотное сведение
